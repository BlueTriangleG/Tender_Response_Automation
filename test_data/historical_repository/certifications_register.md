# Certifications Register Excerpt

## Current approved claims

- ISO 27001 certified
- SOC 2 Type II report available for the core platform

## Restricted claims

- FedRAMP High authorization is not currently an approved claim in the response library.
- IRAP PROTECTED certification is not currently an approved claim in the response library.
- Where a client requests one of the above, the answer must state that the certification cannot be claimed based on the current approved material and should be referred for human review.
